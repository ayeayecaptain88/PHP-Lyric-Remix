PHP Lyric Remix 

A fun and interactive PHP project that dynamically generates lyrics for the song **"Just the Way You Are" by Bruno Mars**. This project demonstrates the use of PHP variables, arrays, loops, and string interpolation to remix song lyrics and display them nicely on a webpage.

Features

- Dynamically generates verses, choruses, and bridge sections using PHP arrays.
- Utilizes variables for nouns, verbs, and adjectives to remix lyrics.
- Styled with a feminine and love-themed design using CSS:
  - Soft gradient background
  - Floating shapes for decoration

Technologies Used

- PHP – for dynamic content generation
- HTML5 – structure and semantic elements
- CSS – for custom styling and aesthetic design
